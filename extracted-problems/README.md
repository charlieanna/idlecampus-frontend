## Problem Format

### Algorithm & Data Structure Problems Include:
- Problem ID and title
- Difficulty level (Easy/Medium/Hard)
- Concept and family tags
- Detailed description
- Key insights
- Examples with explanations
- Constraints
- Hints
- Starter code (Python/JavaScript)
- Solution code with explanations
- Time/Space complexity analysis
- Test cases (including performance tests)

### System Design Problems Include:
- Problem ID, title, and category
- Difficulty level (Easy/Intermediate/Advanced, L5-Staff, L6-Principal)
- Goal and description
- Functional requirements
- Non-functional requirements (latency, throughput, scalability, etc.)
- Constants and assumptions
- Available components (palette)
- Tier checkpoints
- Reference solution architecture
- Comprehensive solution analysis:
  - Phase analysis (normal, peak, failure scenarios)
  - Trade-offs and alternatives
  - Cost analysis (compute, storage, network)
  - Scaling strategies
  - L5/L6 specific considerations (migration, organizational impact, research foundations, novel algorithms)
- Tutorial narratives (for tutorial problems)
